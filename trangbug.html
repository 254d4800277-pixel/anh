Trang Phân Tích Lỗi Sai (Debug)

Nhập nguyên văn đoạn mã lỗi đề bài cho vào ô bên trái, sửa lại cho đúng vào ô bên phải kèm chú thích giải thích lỗi sai.

❌ Đoạn mã lỗi gốc (Đề bài cho): 
function addCourse(newCourse, list) {
  if (mssv % 2 !== 0) { 
    list.push(newCourse);
  } else {
    list.unshift(newCourse);
  }
  renderTable(list);
}
✔ Đoạn mã chuẩn (Đã chỉnh sửa): 
function addCourse(newCourse, list, mssvValue) {
  const lastDigit = parseInt(mssvValue.slice(-1));
  
  if (lastDigit % 2 !== 0) { 
    list.unshift(newCourse); // Lẻ -> Chèn ĐẦU bảng
  } else {
    list.push(newCourse);    // Chẵn -> Chèn CUỐI bảng
  }
  renderTable(list);
}
🔍 Giải thích chi tiết các lỗi sai được phát hiện:
  • Lỗi tầm vực (Scope Error): Hàm gốc sử dụng trực tiếp biến mssv nhưng biến này chưa hề được định nghĩa hay truyền từ tham số của hàm, làm chương trình báo lỗi crash hệ thống.
  • Lỗi ngược logic đề bài: Đề bài yêu cầu số lẻ chèn lên đầu bảng (`unshift`), số chẵn chèn xuống cuối bảng (`push`). Đoạn mã gốc lại đi dùng push khi điều kiện là số lẻ % 2 !== 0 $\rightarrow$ Hoàn toàn làm ngược so với đề bài.
