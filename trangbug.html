Trang Phân Tích Lỗi Sai (Debug)

Nhập nguyên văn đoạn mã lỗi đề bài cho vào ô bên trái, sửa lại cho đúng vào ô bên phải kèm chú thích giải thích lỗi sai.

❌ Đoạn mã lỗi gốc (Đề bài cho): 
function addCourse(newCourse, list) {
  if (mssv % 2 !== 0) { 
    list.push(newCourse);
  } else {
    list.unshift(newCourse);
  }
  renderTable(list);
}
✔ Đoạn mã chuẩn (Đã chỉnh sửa): 
function addCourse(newCourse, list, mssvValue) {
  const lastDigit = parseInt(mssvValue.slice(-1));
  
  if (lastDigit % 2 !== 0) { 
    list.unshift(newCourse); // Lẻ -> Chèn ĐẦU bảng
  } else {
    list.push(newCourse);    // Chẵn -> Chèn CUỐI bảng
  }
  renderTable(list);
}
